- Modern and intuitive user interface
- Product catalog with images
- Category-based filtering

## Tech Stack

### Frontend
- Next.js 14
- React.js
- TypeScript
- Tailwind CSS
- Next.js Image Optimization

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- TypeScript
- CORS enabled
